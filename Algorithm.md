
## Thresholds
- Safe: width < 0.3mm (depth < 20mm)
- Warning: 0.3mm < width < 1mm (depth 20mm - 40mm)
- Critical: width > 1mm (any depth)

- the engineer determines if it's a "structural shear stress" crack or not
- the engineer determines the depth manually
